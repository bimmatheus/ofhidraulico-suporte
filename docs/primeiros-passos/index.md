# Primeiros passos

A porta de entrada do OFHidraulico. Tudo o que você precisa fazer **uma vez por
projeto** antes de começar a modelar e dimensionar.

## Artigos

<div class="grid cards" markdown>

-   :material-cog:{ .lg .middle } __[Configurações gerais](configuracoes-gerais.md)__

    O assistente principal: 4 abas (**Geral, Sistemas, Dimensionamento, Aviso**)
    onde você define unidades, vincula sistemas do Revit, configura rugosidade
    por material e ajusta limites de aviso.

</div>

## Em preparação

- *Instalação e ativação do plugin*
- *Transferindo famílias do template oficial*
- *Importando configurações de outro projeto*

## Fluxo recomendado para projeto novo

```
1. Configurações gerais       ← define base do projeto
2. Transfer families          ← carrega famílias OFH
3. Equipment Manager          ← configura peças (pesos, vazões, comprimentos eq.)
4. Modelar a rede no Revit    ← tubos, peças, conexões
5. Gerenciador de sistemas    ← calcula, valida, exporta
```

## Pré-requisitos

- [ ] Revit instalado (versões 2021 a 2027 suportadas)
- [ ] OFHidraulico instalado e ativado
- [ ] Login na conta Orçafascio (necessário em primeira instalação)
