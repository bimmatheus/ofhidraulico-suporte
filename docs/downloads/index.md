# Downloads

Materiais técnicos em PDF para download.

!!! info "Como funciona"
    Os PDFs são committados neste repositório e atualizados sempre que o time
    publica nova versão dos materiais. Para sincronizar localmente a partir do
    repositório do plugin, rode `scripts/copy-pdfs.sh`.
