# Documentação e pranchas

Comandos e janelas que ajudam você a **documentar** o projeto: etiquetar
elementos, duplicar vistas para pranchas e inspecionar propriedades das peças.

## Artigos

<div class="grid cards" markdown>

-   :material-information-outline:{ .lg .middle } __[Painel de propriedades (OFH)](painel-de-propriedades.md)__

    Janela flutuante (modeless) que mostra e edita propriedades OFH da instância
    selecionada. Útil para configurar bombas e hidrômetros no fluxo de modelagem.

</div>
